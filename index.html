    Checking & Switching
    
					
    Default (different in different browsers)
    
					
					
					
    Styled
    
					
					
					
    
					
					
					
					
    
					Syntax:
					
    <label class="is-[type]">
	<input type="[type]" />
	<span></span>
</label>
    
					
    If you want a cross or dash, set <label class="is-checkbox" data-is-icon="[x or -]"> (for
						checkboxes only). Span
						is empty but it is very important! Type can be radio or checkbox
    
					
					
					Syntax:
					
    <label data-toggleswitch="[type]">
	<input type="chekbox" />
	<span></span>
</label>
    Type can be square (default) or round
					
    3D switch
					
					
					
    Syntax:
					
    <label class="is-3Dswitch">[heading]
	<input type="checkbox" />
	<span>[checked]</span>
	<span>[unchecked]</span>
</label>
    
					
    Dropdown
    
					
    Default
    
					
					
    Styled
    
					
					Syntax:

	<select>

    Wait the minute. I don't want these colors!
    
					You don't have to edit source code. You can do:
					
    body {
	--is-bgcolor: #0FF; /*default background, progress value, slider thumb color; 3D switch checked; default: #2196F3*/
	--is-unchecked: #DDD; /*unchecked input, slider; profress bar, unselected side in 3D switch ; default: #CCC*/
	--is-color: #DFF; /*toggle switch button, dropdown and checkmark color; default: #FFF*/
}
    
					
					
    And where is colour of unchecked 3D switch?
    
					
    There is no custom property. But you can do:
    
					
    <label class="is-3Dswitch">[heading]
	<input type="checkbox" />
	<span>Yeap</span>
	<span style="background: #F00">Nope</span>
</label>
    
					If you want to change the color of placeholder (element must have class is-text) add custom
					property --is-placeholder-color: #FFF;
					
    Note: Microsoft browser except for Chromium Edge don't support this.
